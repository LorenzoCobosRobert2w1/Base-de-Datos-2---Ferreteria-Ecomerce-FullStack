# RESTRICCIONES TÉCNICAS Y DE EJECUCIÓN

- **Stack Backend:** Debes usar exclusivamente JavaScript y MongoDB.
- **Stack Frontend:** Utiliza únicamente HTML, CSS y JavaScript puro (Vanilla JS). Está estrictamente prohibido el uso de frameworks pesados en esta etapa.
- **Diseño UI:** La interfaz visual debe tener un estilo moderno y minimalista.
- **Regla de Generación:** NO generes todo el código del proyecto de una sola vez. Limítate a entregar el plan técnico desglosado en tareas pequeñas y manejables. El código se solicitará iterativamente más adelante.
